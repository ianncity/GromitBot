# GromitBot - World of Warcraft Pixelbot for Turtle WoW

## Requirements
- Python 3.8+
- pyautogui
- numpy
- opencv-python
- pillow
- requests
- discord.py
- aiohttp
- pynput
- keyboard
- mouse

## Installation
Run `install.bat` to install all dependencies.

## Features
- Fishing bot with herbalism gathering
- Leveling bot with pathfinding
- Inventory management
- Human-like behavior (random delays, pauses, mouse movements)
- Discord command integration
- Crash recovery with state persistence

## Usage
1. Run `start.bat` to start the bot
2. Configure settings in `config.json`
3. Use Discord to send commands (if enabled)
